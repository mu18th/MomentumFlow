# Smart Task Manager (Kanban Board)

This project is a simple **Smart Task Manager** built using **HTML**, **CSS**
It allows you to:
- Add new tasks with descriptions and priorities.
- View tasks in a **Kanban board** style: **To Do**, **In Progress**, and **Done**.
- Organize your workflow visually.

## Technologies Used
- HTML5
- CSS3
